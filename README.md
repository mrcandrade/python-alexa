Olá, para pesquisa com chatgpt, introduza sua chave token api da openai na linha 12, entre as aspas [openai.api_key = 'AQUI'] 
Comandos disponíveis na primeira versão:
-Solicitar o horário -> deve conter a palavra "horas" na entrada com microfone. 
ex: (que _horas_ são)
-Realizar pesquisa com wikipedia -> deve conter a palavra "pesquisar" seguido do objeto/pessoa em questão 
ex: (pesquisar Brasil - resposta wikipedia -)
-Correção de palavras -> deve conter "como escrever" ou "corrigir" na entrada com microfone. 
ex: como escrever paralelepípedo)
-Realizar pesquisa com chat gpt -> deve conter "gpt" na entrada com microfone, seguido da entrada que será introduzida ao GPT. 
ex: (gpt como posso iniciar meus estudos em python?)
